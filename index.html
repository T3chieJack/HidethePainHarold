Image Replacer Tool

Generate the files needed to create a browser extension for replacing images dynamically!

How It Works

This tool lets you generate JavaScript, CSS, and a manifest file to create a Chrome extension. Follow the instructions below.

Instructions

Instructions for Using the Image Replacer Tool Enable Developer Mode: Open Google Chrome and navigate to chrome://extensions. In the top-right corner, toggle the switch labeled Developer mode to the ON position. Download the Files: From this website, download the following files: JavaScript File (image-replacer.js) Manifest File (manifest.json) Optional CSS File (style.css) Create a New Folder: Create a new folder on your computer, for example: ImageReplacerExtension. Add the Files: Place the downloaded files (JavaScript, Manifest, and CSS) into the folder you created. Load the Extension: Return to chrome://extensions. Click the Load unpacked button. In the file dialog that opens, select the folder you created in step 3. Test the Extension: Open any website with images. You should now see the images being replaced with ones from the tool automatically!

JavaScript Code

function replaceImages() {
  const allImages = document.getElementsByTagName("img");
  for (let i = 0; i < allImages.length; i++) {
    const imageNumber = (i % 10) + 1;
    allImages[i].src = \`https://t3chiejack.github.io/Images/\${imageNumber}.png\`;
  }
}

const observer = new MutationObserver(replaceImages);
observer.observe(document.body, { childList: true, subtree: true });
replaceImages();
